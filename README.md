# 📊 Amazon Review Sentiment Analysis {Brainwave matrix intern}

## 🚀 Features
- Text preprocessing & cleaning  
- Sentiment score extraction  
- Visualization of sentiment trends  

## 🛠️ Tech Stack
- Python, Flask  
- NLTK, Scikit-learn  
- Matplotlib, Seaborn
